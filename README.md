# 🏹 O Veredito - Portal Interativo de Cultura Indígena

Este projeto foi desenvolvido como Trabalho de Conclusão de Curso (TCC) na **Escola Yervant Kissajikian**, no curso técnico de Análise e Desenvolvimento de Sistemas. O objetivo é oferecer uma experiência imersiva e educativa sobre a história e as narrativas dos povos originários.

## 🚀 Diferencial Técnico: Desenvolvimento Assistido por IA
Neste projeto, utilizei ferramentas de **Inteligência Artificial Generativa** para acelerar a curva de aprendizado e implementação de bibliotecas complexas. Atuei como arquiteto da solução, orquestrando a lógica gerada pela IA para garantir a integração fluida entre um front-end de alto impacto visual e um back-end robusto.

## 🛠️ Tecnologias Utilizadas

### Back-end
* **Java 17:** Linguagem principal focada em Programação Orientada a Objetos (POO).
* **Spring Boot:** Framework utilizado para gerenciar a lógica de processamento e captura de dados.
* **Sistema de Feedback:** Implementação funcional para captura de dados de formulários e processamento no servidor.

### Front-end Imersivo
* **Three.js:** Utilizado para a criação de elementos e portais 3D interativos.
* **GSAP (GreenSock Animation Platform):** Responsável pelas animações fluidas e transições de cena.
* **HTML5, CSS3 e JavaScript (ES6+):** Estrutura e interatividade base da plataforma.

## 🌟 Funcionalidades Principais
* **Navegação Imersiva:** Fluxos que simulam fatos históricos e narrativas indígenas.
* **Alta Performance:** Otimização para garantir uma experiência fluida ao usuário final, mesmo com elementos 3D.
* **Sistema de Feedback:** Captura e tratamento de dados enviados pelos usuários diretamente no back-end em Java.
